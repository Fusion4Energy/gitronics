# Getting Started

Choose the path that matches where you are:

1. **Migrate an existing model**: convert a monolithic MCNP input into a Gitronics project layout.
2. **Build a model**: assemble an MCNP input from an existing Gitronics project.
3. **Start a new project**: create a fresh project structure manually.

## Typical workflow

If you are converting a legacy model:

1. Run migration.
2. Review generated envelope and filler files.
3. Run a build to verify the round-trip.

If you are starting from scratch:

1. Create the project folders and baseline configuration.
2. Add envelope/filler/data-card files.
3. Build with the baseline configuration.
